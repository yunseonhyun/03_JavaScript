함수(function)

        - 목표하는 결과를 만들기 위한 코드의 묶음

        - 코드를 {} (블록)으로 묶어 작성하고
            붙여진 이름을 호출해서 사용
        * 함수 선언
        * 함수 정의
        * 함수 호출

        [기본 함수 형태]

        function 함수이름() { // 함수 선언
            // 함수 정의
        }

        함수이름(); // 함수를 단독으로 호출할 수 있다.

매개변수(Parameter), 전달 인자(Arguments)

        * 매개 변수(Parameter)
        - 함수 선언 시 ()안에 작성 되는 변수
            -> 전달 인자로부터 받은 값을 저장
            -> 함수 정의 부분에서 사용 가능

        * 전달 인자(Arguments)
        - 함수 호출 시 () 안에 작성되는 값
            -> 함수 매개 변수로 전달되는 값

return(반환)

        - 현재 수행중인 함수를 종료하고 호출한 곳으로 돌아감
            -> 함수 호출 구조는 Stack 형식으로 되어있다.
            -> FILO(First In Last Out, 선입후출) 구조
        - 호툴한 곳으로 돌아갈 때 return 옆에 작성된 값을 가지고 돌아갈 수 있다.

        ex) return 100; // 호출한 곳으로 100을 가지고 돌아감


        - 함수 마지막에 return 구문이 생략되어 있다면
            자동으로 return; 구문이 추가된다.

익명 함수(이름이 없는 함수)

        [기본 함수]
        function 함수이름(매개변수) {
            // 함수정의
        }
        함수이름(); 이름이 있으니까 함수이름으로 호출 가능
        **********************************************
        [익명 함수]
        function([매개변수]) {
            // 함수 정의
        }
        // 함수 이름이 없어서 단독으로 호출할 수 없다!!!

        상황에 주로 사용

        ex) 요소.addEventListener("click", function() {

        })

        ex) const 더하기 = function(a, b) {

            return a +b;
        }

화살표 함수(Arrow Function)

        - 익명 함수의 한 종류
        - 익명 함수를 간단히 표기하는 방법(ES6)

        익명 함수 : function(){}
        
        1. 기본 형태 : () => {}

        
        2. 매개변수 1개 : (e) => {}
                          e  => {}       // () 생략 가능
        3. 매개변수 n개 : (a, b, c) => {} // () 생략 불가
        4. 함수 정의 부분에 return 한 줄만 있을 경우 : 
            1) 익명 함수 :              function(){return 1 + 2;}
            2) 화살표 함수(기본) :      () => {return 1 + 2;}
            3) 화살표 함수(return 한줄) : () => 1+2 //{return} 생략 가능
        5. 함수 정의 부분에 return 한줄만 있는데
            return 값이 object(객체, 배열 등) 인 경우
            {return} 생략 불가 !!
            () => {return {name : "홍길동", age:20}}
