객체(Object)

        - 속성과 기능으로 이루어진 것
            -> 속성 == 변수(필드)
            -> 기능 == 메서드(객체에 작성된 함수)

        - {} (블록) 내에 Key, Value가 모여있는 형태
            -> Key == 변수명/메서드명 (중복x)
            -> Valeu == 값, 함수 

        **** K:V 가 모여있는 형식 == Map (맵) ****

        - JS 객체 작성법    
        {K:V, K:V, K:V, ...}

        [JS 객체 생성 방법]
        1. {} : JS 객체 리터럴 표기법을 이용한 생성

        2. 생성자 함수 : 생성자 + new 생성자()를 이용한 생성

        ** 리터럴 == 값 **
        - value 대신에 리터럴이라는 명칭을 사용

        ex)
        문자열(string) -> "",''
        숫자(number) -> 1, 2.34, +50, -50
        배열(array) -> []
        객체(object) -> {}
        논리(bollean) -> true, false
