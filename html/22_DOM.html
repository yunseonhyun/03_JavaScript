DOM(Document Object Model)

        HTML(웹문서, 웹페이지) 문서를 객체 기반으로 표현한 것
        -> HTML 문서에 작성된 내용을 트리구조로 나타냈을 때
           각각의 태그, TEXT, COMMENT를 NODE라고 한다.

           * node : document(문서) 트리구조를 구성하는 모든 개별 요소를 가리키는 기본 단위
                    요소 = element(태그와 태그내부에 작성된 텍스트나 value까지 포함하는 것을 의미)
                    웹 페이지를 구성하는 모든 것
                        -> HTML 태그 
 
  등 ..
                           태그 안에 텍스트(글자)
                           주석
                           코드에 작성된 모든 내용은 각각의 node라고 불림
                    노드 내부에서 태그와 요소로 분리됨!!
                    nodeJS : 프론트엔드 코드로 작성되는 모든 것에 대한 JavaScript 설정을 나타냄
                            nodeJS로 백엔드 구성이 가능한 사람 : javaScript 언어기반 백엔드 기술자
                            nodeJS로 프론트엔드 구성이 가능한 사람 : javaScript 언어기반 프론트엔드 기술자
                            (* 기술도구는 nodeJs를 활용하지만 언어의 요구사항이 js가 아니라 ts나 다른 언어일 수는 있음)
