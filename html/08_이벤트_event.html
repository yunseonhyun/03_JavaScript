이벤트 e, event, value, target

        1. e와 event는 같은 것
        e와 event는 모두 이벤트 객체
        
        function 기능1번(e) {
            // 기능 1번에 해당하는 기능 내용들
        }
        function 기능2번(event) {
            // 기능 2번에 해당하는 기능 내용들
        }
예제 1: e와 event는 동일하다.
            function 클릭처리(e) {
                console.log("e 사용 : ", e.type);
            }

            function 클릭처리2(event) {
                console.log("event 사용 : ", event.type);
            }
