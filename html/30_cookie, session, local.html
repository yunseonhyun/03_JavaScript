웹 스토리지 - 쿠키, 세션, 로컬

쿠키(cookie)

서버와 주고 받는 작은 데이터 로그인 정보 저장에 많이 사용 -> 직접적으로 로그인 정보를 저장하는 것이 아니라 로그인을 할 수 있는 토큰(=인증키)을 저장 후 로그인 정보를 가져올 수 있도록 설정

세션(session)

탭을 닫으면 사라지는 데이터, 임시 저장용으로 많이 사용 로그인 정보 저장에 많이 사용 -> 쿠키처럼 토큰(=인증키) 저장하는 방식이 아닌, 아이디와 같은 정보를 인식하여 탭을 닫거나, 특정 시간이 만료되면 세션에 저장되어 있는 정보를 자동으로 삭제

로컬(local)

브라우저를 닫아도 남아있는 데이터, 설정값 저장에 사용 즐겨찾기와 같은 용도로 많은 사용
